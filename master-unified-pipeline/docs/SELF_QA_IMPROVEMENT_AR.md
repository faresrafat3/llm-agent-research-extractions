# أسئلة تحسين ذاتية — في نطاق هدف المشروع

**الهدف الثابت:** استخراج أنظمة LLM agent → دقة وعمق → رسوم EN/AR → دمج في أرسينال/المجمّع → مستودعات عامة.

لا نوسّع لنحو منتج runtime أو أبحاث جديدة خارج هذا الهدف.

## مبدأ التحسين المعتمد

> **أفضل تحسين داخل هدفنا = إغلاق فجوات التكامل بين ما استُخرج وبين أرسينال/المجمّع، لا تكديس أوراق جديدة قبل الربط.**

ترتيب القيمة:
1. ربط ToT → ARSENAL L3  
2. خريطة قرار ToT vs LATS  
3. تحديث المخططات والفهارس والمعيار  
4. (لاحقاً) Voyager / OPRO كاستخراجات جديدة
