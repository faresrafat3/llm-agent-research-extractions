# معيار الجودة الموحد

كل مشروع داخل هذا الريبو يجب أن يحتوي على:

- README واضح.
- ملخص بحثي أو system summary.
- prompts extraction.
- logic/flow extraction.
- Mermaid graph إنجليزي وعربي.
- deep dive matrix.
- final completeness check عربي.
- quality review عربي.

تم تحديث المشاريع الثلاثة لتتبع هذا المعيار قدر الإمكان، مع احترام اختلاف بنية كل مشروع أصلي.
