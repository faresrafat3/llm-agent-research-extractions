## What this repo contains

A high-quality extraction/audit package for Self-Refine:

- research summary,
- complete prompt extraction,
- raw prompt files,
- Python logic/flow extraction,
- Visual Self-Refine GPT-4V notebook audit,
- docs/website prompt audit,
- deep-dive task matrix,
- paper appendix notes,
- Mermaid graphs in English and Arabic,
- Arabic final audit and completeness checklist.
