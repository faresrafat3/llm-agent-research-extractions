# مسح الذكاء العلمي — مخطط المنطق بالعربية

المصدر: `graph_arabic.mmd`

## السرد

معمارية نموذجية شكل1: استعلام المستخدم نص+بيانات → المخطط يفكك المهمة إلى مهام فرعية يسترجع سياق معرفة من الذاكرة ينفذ إجراءات عبر مساحة الإجراءات واجهات برمجة محاكيات أدوات مختبر محركات بحث النموذج نفسه كاستدلال داخلي → نتائج وسيطة يفحصها المتحقق دقة اتساق معقولية علمية → نتائج موثقة تخزن في الذاكرة تحسين قرارات مستقبلية → إذا يشير التحقق إلى مزيد إجراءات تصحيحات المخطط يولد خطط جديدة يستدعي مساحة إجراءات تكراري حتى يؤكد المتحقق مخرجات تلبي معايير صلاحية قابلية تكرار → نتيجة متكاملة نهائية تعود للمستخدم. إدراك متعدد وسائط قدرة جوهرية للمخطط.

عائلات المخطط:
- P1 تعليمي/مخطط-مدفوع: قوالب سير عمل محددة ReAct فكرة-إجراء-ملاحظة مخططات أدوات إرشادات مجال، نظام شخصية خبير بطاريات مخطط إجرائي 1 تصميم بلورة→2 فحص DFT→3 تخطيط تخليق→4 اختبار كهروكيميائي قائمة أدوات VASP XRD_Simulator Autolab_API قيود تجنب Co هدف >4V
- P2 معزز سياق: ترميز سجلات تاريخية بحثية سياق مبحوث موجهات، تاريخي NMC811 فشل @400 دورة تلاشي LFP مستقر 160، معرفة هدف موصلية sigma>1000 جهد 4.0-4.3V، مكتبة قوالب محيط أدوات ذاتي تطور سياق مستمر
- P3 تداولي/تأملي: متعدد الأدوار توليد→نقد→مراجعة موجهات وصفية تقييم اتساق منطقي جدوى تحديد أنماط فشل مراجعة، تفكير سلسلة أفكار تأمل ذاتي، إعادة تخطيط مدفوعة بخطأ، مراحل تأمل صريحة، تأمل قائم على نموذج رؤية، مثال توليد أولي أكسيد غني ليثيوم تأمل عيوب دورات 480 فشل أمان إطلاق O2 مراجعة منشط Mg طلاء Al2O3 تأمل 550 نجاح تقارب خطة نهائية
- P4 مبني على بحث: استكشاف فوق فضاءات خطط توليد تقييم مرشحين متعددين اختيار أمثل شجرة فكر MCTS بحث شعاعي عقد خطط جزئية حواف توسعات تقييم إرشادي نماذج قيمة متعلمة تغذية كيمياء كوانتية جودة شكل نموذج رؤية توسيع انتقائي عالي درجات تقليم منخفض جودة، ChemReasoner شجرة هرمية خطط استعلام نوع حفاز معايير تضمين/استبعاد عوامل علائقية تغذية كيمياء كوانتية طاقات امتزاز حواجز تفاعل استقرار مكافآت تقليم، AI Scientist-v2 بحث شجري تجريبي مدير تنسيق توسع 4 مراحل تحقيق أولي ضبط فرط معاملات تنفيذ أجندة بحث دراسات استئصال متوازي جودة شكل، HE-MCTS فصل تخطيط أدوات نموذج سياسة تنفيذ نموذج تنفيذ، جذر كاثود >200 فروع scores 0.5 0.7 0.65 0.82 0.75 DFT E_cal -3.1 eV دورات 520 مسار مكافأة أقصى
- P5 تفاعلي أدوار/متعدد وكلاء: توزيع تخطيط عبر مجموعات متعاونة وظائف متخصصة اقتراح عيب جدوى تغذية ديناميكيات فريق علمي نقاش تكراري توافق مناظرة بطولة تصويت متعدد جولات توليد-تأمل مقترن نقاد متخصصون تنوع جدوى صرامة علمية مطور ناقد حكم مراجعة وصفية مشرف، مصمم مواد ناقد أمان مهندس تخليق نقاش تقييم D:✓ C:✗ E:? خطة توافق مراجعة غني Mn LiNiO2-Mg
- P6 برمجي كود/DSL/DAG: تمثيلات خطة قابلة تنفيذ آليًا مولد كود LLM توليد خط أنابيب DFT+تخليق كسكريبت Python → قطعة قابلة تنفيذ workflow=DSL_Pipeline().add(DFT_Screening) أو DAG تبعيات مهام صريحة
- L1 SFT مدرب مجال: AstroMLab BioGPT Chemma LLaMA-2-7b 34B tokens تخليق عكسي DrugAssist إلخ استيعاب استراتيجيات تخطيط تدريب مسارات خاصة مجال
- L2 RL محسن تفضيل: BioScientist Agent CheMatAgent Chemma CycleResearcher ReFT STEP-DPO Flow-DPO Sci-MARL MolRL-MGPT إلخ إشارات مكافأة

ذاكرة 5 أنواع: M1 عاملة سياق scratchpad فكرة-إجراء-ملاحظة، M2 عرضية Reflexion تأملات لفظية تاريخ تجارب دروس آخر K، M3 دلالية قاعدة معرفة متجهة KG HoneyComb مكتبة قوالب محيط أدوات ذاتي تطور، M4 إجرائية مكتبة مهارات Voyager وصف مهارة بحد أقصى 6 جمل نص واحد استرجاع تشابه، M5 هجين تركيب هرمي

مساحة إجراء 5 أنواع: A1 استدلال داخلي LLM نفسه سلسلة أفكار، A2 أداة واجهة 18+ ChemCrow VASP XRD_Simulator Autolab_API GeneGPT PaSa Semantic Scholar إلخ إجراء ToolName[args]، A3 تنفيذ كود Python DSL ORGANA وكلاء فرعيون فهم تحسين حسابات كيميائية ترتيب قوارير، A4 محاكاة DFT تغذية كيمياء كوانتية محاكيات بطارية طاقات امتزاز حواجز استقرار GeoSim.AI OpenFOAMGPT إلخ محاكاة نتائج كل فرع توسع فقط معقول فيزيائيًا باستخدام تغذية توجيه SGA، A5 مختبر فيزيائي Coscientist مختبر روبوتي Suzuki Sonogashira AutoLabs معالجات سائلة عالية إنتاجية تصحيح ذاتي تحويل تعليمات لغة طبيعية بروتوكولات قابلة تنفيذ ORGANA لغة طبيعية كيميائي توضيح

متحقق 4 أنواع: V1 نقد ذاتي سلسلة أفكار تأمل ذاتي مراحل نقد صريحة جودة شكل نموذج رؤية، V2 قائم على أداة تغذية محاكاة طاقات امتزاز ذرية حواجز استقرار مكافآت تقليم اختبارات كود تسجيل إرشادي تتبع خطأ شامل GeoAgent، V3 بشر في الحلقة إشراف خبير طيف إشراف مستمر تدخل انتقائي قائم على استثناء طرائق بوابات موافقة بروتوكولات تجريبية سلامة حرجة إيقاف عرض إجراءات تخليق تسلسلات تحكم روبوتية عمليات خطرة تنتظر موافقة بشرية صريحة، تغذية تقييم مخرجات بحثية خبراء مجال تقييم جودة حداثة صلاحية نقد نوعي صقل، تكرار تعاوني بشري-ذكاء حوارات متعددة توجيه قيود تصحيحات، تدخل تصحيح أخطاء فشل تصحيح ذاتي تحرير يدوي كود ضبط معاملات إعادة توجيه، أمثلة مختبر الوكيل Schmidgall أبحاث تعلم آلي تغذية توجيه كل مرحلة، StarWhisper فلكيون عملية تلسكوب، ORGANA REASONER يطلب مستخدم تحقيق قرار مزيد إجراءات عدم تطابق نتائج، BIA تدخل بشري مفاصل حرجة تقسيم مجموعة فرعية لا غنى عنه، ChemCrow لوحة 4 خبراء كيمياء صحة جودة استدلال درجة إكمال إصلاح إجراءات غير صالحة قبل روبوت، dZiner مراجعة كيميائي، Chemma تعلم نشط تغذية تجربة رطبة، MAPPS علماء تقييم تصنيف صقل موافقة صريحة قبل محاكيات مكلفة، MatPilot تعاون بشري-آلة؛ V4 نقد متعدد وكلاء لعب أدوار تصويت متعدد جولات نعم/لا تعديل تكراري حتى توافق أو محاولات قصوى، VirSci صقل فكرة منظمة ناقد حداثة جدوى تأثير منخفض مفلتر عالي مصقول، CellAgent مقيم منفذ ضبط فرط معاملات اختيار أداة تلقائي scRNA-seq، Sparks توليد-تأمل مقترن وضوح حداثة جدوى صحة تقنية اكتمال التزام، AccelMat نقاد متخصصون تنوع جدوى صرامة علمية، STELLA مطور بيئة بناء إنشاء كود تدريب نموذج كتابة تقرير ناقد تقييم وسيط عيوب تغذية قابلة تنفيذ، AtomAgents ناقد تحقق قائم على دور اكتمال صحة، AutoLabs مشرف وكلاء فرعيون فهم تحسين حسابات كيميائية ترتيب قوارير خطوات معالجة خطوات نهائية فحوص ذاتية تحقق نهائي، AI co-scientist مناظرة بطولة 4 مناظرة 2 مع 2 ضد حكم مراجعة وصفية تركب رؤى جولات متعددة أنماط متكررة تحسين
