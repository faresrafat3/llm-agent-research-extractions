# GAPMAP — مخطط المنطق بالعربية

المصدر الكامل: `graph_arabic.mmd`

## نظرة عامة

النظام يتكون من مسارين رئيسيين: الفجوات الصريحة والفجوات الضمنية، مع مرحلة تحضير.

1. **التحضير**: تحميل CSV → إزالة التكرار في المقتطفات → تقسيم باستخدام Stanza محاذاة جملية ≤1000 كلمة (بديل regex) مع الحفاظ على تماسك الجملة.

2. **المسار الصريح zero-shot**:
   - لكل chunk → استدعاء LLM بنظام `أرجع JSON فقط` + قالب `PROMPT_TEMPLATE` يتطلب جملة الفجوة الحرفية، الجمل الداعمة، التبرير، كلمات الإشارة للجهل.
   - استخراج JSON بال regex، إزالة أسوار الكود وعلامات التفكير، التحقق من المفاتيح المطلوبة.
   - تجميع النتائج، إزالة التكرار عبر مطابقة الجملة بعد ضغط المسافات.
   - الحفظ لكل معرف JSON.
   - تقييم IPBES: ROUGE-L F1 مع stemming مطابقة واحد لواحد عتبة 0.55 → دقة، استدعاء، F1.
   - تقييم COVID: تصفية الفجوات غير البحثية، التحقق مقابل قاموس إشارات الجهل (يجب احتواء إشارة واحدة على الأقل) → الدقة + تداخل Venn + مخطط عنكبوتي فئات.

3. **المسار الضمني TABI**:
   - مجموعة يدوية 212 فقرة من 137 مقال PubMed، تم إخفاء اتجاهات مستقبلية في النهاية.
   - موجه 3-shot مع أمثلة (فأر→إنسان فجوة ترجمة، اصطناعي→حقيقي حاجز، مجموعات متضاربة شذوذ) → ينتج Claim، Grounds (مقتبسة)، Warrant (جملة واحدة)، Bucket (أكثر احتمالاً مقابل أقل احتمالاً)، Category (5 أنواع).
   - التقييم عبر RoBERTa-large استدلال ثنائي الاتجاه عتبة 0.4 للمطالبة مقابل الذهب والمبرر مقابل المقدمات؛ النجاح إذا طابق أي مرشح الذهب.
   - تجربة النص الكامل: 24 ورقة 19 مجالًا → GPT-4o متعدد الوسائط سياق طويل → 5-10 فجوات منظمة كل مع أدلة ومبرر ونوع واتجاه مستقبلي وملاحظات جدوى و bucket → استبيان 18 مؤلف نسبة اتفاق.

4. **مقارنة النماذج**: 7 نماذج (GPT-5، GPT-4o، mini، Gemma-9B، Llama 70B/17B MoE/8B) × إعدادين (بدون حد مقابل 1K chunks).

انظر ملف Mermaid للتدفق القابل للتنفيذ.
