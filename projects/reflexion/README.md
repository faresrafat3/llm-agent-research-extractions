# Reflexion — Complete Prompt, Logic, Flow, and Graph Extraction

Paper: https://arxiv.org/abs/2303.11366  
Source code: https://github.com/noahshinn/reflexion  
Audited commit: `218cf0ef1df84b05ce379dd4a8e47f17766733a0`

## What this repo contains

A high-quality extraction/audit package for Reflexion:

- research summary,
- prompt/config/few-shot extraction,
- raw prompt/config files,
- Python logic/flow extraction,
- deep-dive task matrix,
- Mermaid graphs in English and Arabic,
- final Arabic audit and completeness checklist.

- `raw_prompt_files/` — operational prompt/config/few-shot sources.

- `graph_english.mmd`, `graph_arabic.mmd` — Mermaid source graphs.
